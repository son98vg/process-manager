# Zeromq Client-Server example using Python. #

## Overview ##

Demonstrables:

    +   Multiple client architecture.
    +   Fair distribution of work among workers.

## Requirements ##
    +   Zeromq [link](http://www.zeromq.org/)
    +   pyzmq [link](https://github.com/zeromq/pyzmq)

## Run ##
    +   python server.py
    +   python clients.py
